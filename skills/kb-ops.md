# kb-ops

Knowledge base operations — manage project docs across all projects in the OKF.

## Available MCP Tools

| Tool | Description |
|------|-------------|
| `okf_list_projects` | List all projects with file counts |
| `okf_get_project` | Read project profile + agent + status + structure + dependencies + commands |
| `okf_search` | Full-text search across all .md files |
| `okf_get_file` | Read raw content of any OKF file |
| `okf_list_dir` | Browse KB directory structure |

## Usage

1. `okf_list_projects` — discover what projects exist
2. `okf_get_project` — read full project context
3. `okf_search` — find specific content across KB
4. `okf_get_file` — read any specific file
