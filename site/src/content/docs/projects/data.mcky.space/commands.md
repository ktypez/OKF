# Commands

| Command | Action |
|---------|--------|
| `npx vite` | Start dev server |
| `npx vite build` | Production build |
| `npx wrangler pages deploy --project-name data ./dist` | Deploy to Cloudflare Pages |
| `npx wrangler pages dev --port 8788 ./dist` | Run Pages Functions locally |
