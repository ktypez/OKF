# DEC-001: injectManifest PWA strategy

## Decision

Use `vite-plugin-pwa` with `injectManifest` strategy (custom service worker) instead of `generateSW`.

## Rationale

- Fine-grained control over caching strategies per route type
- Offline queue for shift logs needs custom logic
- Cache versioning (ezzy-truck-v3) for rollout control

## Consequences

- Manual SW maintenance: registerSW, cache strategies, shortcuts
- Cache busting on version change requires SW update flow
