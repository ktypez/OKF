# DEC-001: All content follows OKF format

## Decision

All writer output (changelogs, summaries, documentation) must use OKF YAML frontmatter format with proper type, id, and metadata.

## Rationale

- Consistency with workspace knowledge system
- Content is addressable and linkable
- Auto-discoverable by graph indexer

## Consequences

- Every piece of content needs frontmatter
- Requires familiarity with OKF schema
