# DEC-001: Adopt mcky.space neobrutalist design system

## Decision

Replace the 5-theme system with a 2-theme (light/dark) neobrutalist design system from mcky.space.

## Rationale

- Consistency across the mcky.space ecosystem
- Reduced CSS maintenance (one shared design language)
- Self-hosted JetBrains Mono instead of Google Fonts

## Consequences

- BREAKING CHANGE: all theme-related CSS had to be rewritten
- Old theme picker UI replaced with simpler toggle
