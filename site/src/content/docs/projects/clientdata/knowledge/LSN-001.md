# LSN-001: useState hooks spiral — refactor to useReducer

## Lesson

The main `page.tsx` accumulated ~30 tightly coupled `useState` hooks for view state, selection state, form state, and map state. This made it hard to add features without introducing bugs.

## Resolution

Refactor to `useReducer` with a well-defined action type union. Each action maps to a clear state transition.
