# Design System — mcky.space

The authoritative design reference for the mcky.space personal website. Source-of-truth is `src/globals.css` (650 lines); this doc mirrors it so a new route designer can build to spec without re-reading the stylesheet.

## Location
- **File:** `/home/mcky.space/DESIGN.md`
- **Verified against:** `src/globals.css`, `src/layouts/Layout.astro`, `src/components/PageHeader.astro`, `src/pages/blog/[slug].astro` scoped styles

## What it documents
1. **Tokens** — light `[data-theme="light"]` and dark `[data-theme="dark"]` color vars, 8 accents (`--green`/`--amber`/`--red`/`--blue`/`--purple`/`--orange`/`--cyan`/`--pink`), `--border:3px`, `--radius:6px`, hard offset `--shadow`/`--shadow-sm`. Dark theme: `--bg:#0a0a0a`, `--bg-raise:#141414`, `--border:#888888` (deep grey, was navy pre-2026-07-11).
2. **Typography** — JetBrains Mono self-hosted WOFF2, fluid `clamp()` headings, complete working scale (11–15px) per component.
3. **Components** — every selector in globals.css catalogued (53 class selectors + 2 `[data-theme]` + 2 `[data-lang]`): `.neo-card`, `.neo-tag`, `.neo-badge`, `.topnav`, `.sidebar`, `.section`/collapsible, `.stat-block`, `.bar-row`, `.todo-row`, `.chip-group`, `.password-gate` (legacy), `.skel`, `.cursor-blink`, plus the scoped blog-post styles + full highlight.js token palette.
4. **Layout & Grid** — `.app` 560px mobile → full-bleed desktop, `.desktop-layout` sidebar+main, breakpoints 768/1024/1440, `env(safe-area-inset-*)`.
5. **Motion** — `@keyframes shimmer` + `blink`, scan-line reveal + magnetic hover (/) and typewriter (/about) JS recipes, hover micro-affordances, `prefers-reduced-motion` overrides.
6. **Composition recipes** — terminal prompt, neofetch rows, skill bars, project cards, blog list cards, header card, footer, noscript fallback.
7. **A11y** — `:focus-visible`, `:active scale(0.97)`, 43px/33px hit areas, ARIA landmarks, WCAG AAA contrast (18:1 light, 16:1 dark).
8. **Theming persistence** — `data-theme` on `<html>`, FOUC-prevention script in `<head>`, `localStorage.theme`, meta `theme-color`.
9. **Bilingual** — `data-lang="en"` / `"th"`, `.th.md` ↔ `.md` slug pairing, `.lang-toggle` on `/blog/*` only.
10. **Implementation rules** (15) — tokens immutable, dark-theme parity mandatory, self-host fonts only, vanilla JS toggles only, no Alpine/React.
11. **Anti-patterns** — hardcoded hex, dark without light companion, external font CDN, Alpine state for theme, `outline:none` without replacement, animations ignoring reduced-motion, flicker-on-load toggles, trusting `.sec-title.amber/.blue/.purple` class names (all currently set to `--text`).
