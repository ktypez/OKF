# RSK-001: View state lost on hard refresh

## Risk

The SPA uses in-memory React state for the current view (map/list/admin). A hard browser refresh resets this to the default view, losing the user's context.

## Mitigation

- `localStorage` caching of selected client, current view, and map center
- URL hash parameter for view state (planned)
