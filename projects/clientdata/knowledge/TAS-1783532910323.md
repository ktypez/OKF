# TAS-1783532910323: Fix TOCTOU race in POST /api/clients

Severity: High. File: app/api/clients/route.ts lines 37-45. Bug: uploadClientImages runs before duplicate check (getClientById). If ID exists, images upload to R2 but are orphaned. Also, duplicate check + insert are not atomic -- concurrent POSTs can both pass. Fix: Move duplicate check before image upload. Use DB unique constraint and catch insert error, or use a transaction.
