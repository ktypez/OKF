# LSN-001: ReceiptViewer is shared — don't modify independently

## Lesson

The ReceiptViewer component is the canonical receipt template used by both admin and customer (LIFF). Changes must be tested against both surfaces.

## Resolution

Always verify ReceiptViewer changes in both `/` (admin dashboard) and `/liff` (customer view) before committing.
