# DEC-001: POS layout — full-width menu grid + bottom-sheet cart

## Decision

Redesign the POS tab with a full-width menu grid (search + category pills) and cart as a bottom sheet instead of a side panel.

## Rationale

- Better use of mobile screen real estate
- Bottom sheet is more ergonomic for one-handed use
- Category pills + search improve menu navigation speed

## Consequences

- Cart state must persist across bottom sheet open/close
- Bottom sheet height needs to be dynamic based on cart items
